# 坦克大战 - Tank Battle 🎮

经典 **Battle City** 风格坦克大战游戏，纯 HTML / CSS / JavaScript 实现，无需任何外部依赖。

## 快速开始

直接在浏览器打开 `index.html` 即可游玩。

或访问 GitHub Pages: `https://monkeydoor.github.io/tank-battle/`

## 操作

| 按键 | 功能 |
|------|------|
| `W` `A` `S` `D` / `↑` `←` `↓` `→` | 移动坦克 |
| `Space` | 射击 |
| `P` | 暂停 |

## 特性

- 玩家坦克 vs AI 敌人
- 随机生成的地图（砖墙、钢墙）
- 基地（老鹰）保护机制
- 爆炸粒子特效
- 计分、生命、等级系统
- 敌人追击 AI
- 移动端响应式布局

## 规则

- 保护你的基地（黄色旗帜）不被摧毁
- 消灭所有 20 个敌人即可获胜
- 你有 3 条命
- 子弹可被敌方子弹拦截
